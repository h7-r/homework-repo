## 두 정수의 합 구하기

---

## 문제 설명

정수 `num1`과 `num2`가 주어질 때,  
`num1`과 `num2`의 **합을 반환**하도록 `solution` 함수를 완성하세요.

## 제한 사항

- `num1`, `num2`는 정수입니다.
- `-50,000 ≤ num1, num2 ≤ 50,000`

## 입력 예시

- `num1 = 2`, `num2 = 3`
- `num1 = -1`, `num2 = 5`

## 출력 예시

- `5`
- `4`

---

## 문제 해결 단계

### 1단계 : 문제 읽기 (2분)

- 입력 : 정수 `num1`, `num2`
- 출력 : 두 정수의 합
- 조건 : 음수 포함 가능하게

---

### 2단계 : 계획 세우기 (3분)

- 두 정수를 더한다
- 더한 결과를 return한다한다

---

### 3단계 : 구현하기 (10분)

```js
function solution(num1, num2) {
  return num1 + num2;
}
```

### 4단계 : 검증하기 (5분)

- (2, 3) = 5

```js
function solution(2, 3) {
  return 2 + 3;
}
console.log(solution);
```
